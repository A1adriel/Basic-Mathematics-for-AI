# Лабораторная работа №4: Самописный MLP на CPU

## 1. Датасет

**Датасет**: Breast Cancer Wisconsin (Diagnostic)  
**Источник**: UCI ML Repository

### Характеристики:
- **Размер**: 569 образцов
- **Признаки**: 30 числовых признаков ядер клеток
- **Целевая переменная**: Диагноз (0 - злокачественная, 1 - доброкачественная)
- **Распределение классов**: 357 доброкачественных (62.7%), 212 злокачественных (37.3%)

## 2. Предобработка данных

### Разделение данных
- Обучающая: 60% (341 образец)
- Валидационная: 20% (114 образцов)
- Тестовая: 20% (114 образцов)
- Стратификация сохраняет пропорции классов

### Масштабирование
Применяется `StandardScaler()` — все 30 признаков стандартизированы, scaler обучается только на тренировочных данных.

## 3. Архитектура MLP

```
Входной слой:   30 нейронов
Скрытый слой 1: 64 нейрона, ReLU
Скрытый слой 2: 32 нейрона, ReLU
Выходной слой:   1 нейрон,  Sigmoid
```

## 4. Гиперпараметры

```python
learning_rate = 0.01
l2_lambda = 0.001
batch_size = 32
epochs = 300
```

**Оптимальный порог**: 0.3838 (по критерию Юдена, J = 0.9627)

## 5. Результаты

### Метрики на тестовой выборке

| Метрика | Значение |
|---------|----------|
| Accuracy | 0.9561 |
| F1-Score | 0.9650 |
| ROC-AUC | 0.9914 |
| Precision | 0.9718 |
| Recall | 0.9583 |

### Матрица ошибок

```
                  Предсказано
              Злок.    Добр.
Злокачественная  40      2
Доброкачественная 3     69
```

## 6. Анализ обучения

Модель стабилизировалась примерно после 100 эпох, переобучения практически нет.

- Train accuracy: 99.12%
- Val accuracy: 98.25%
- Test accuracy: 95.61%

ROC-AUC = 0.9914 — хорошее качество разделения классов.
